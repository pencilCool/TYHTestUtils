## Installation

TYHTestUtils is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'TYHTestUtils',:git=>'https://github.com/pencilCool/TYHTestUtils.git'
```
